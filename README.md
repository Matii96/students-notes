# Students notes
Presentation project for Node.js and Vue 2 web application.

# Installation
- Server directory:
  ```
  npm i
  npm run build
  npm run db:reset
  ```
  and then remove ./dist
- Client directory:
  ```
  npm i
  ```
# Development:
- Server directory:
  ```
  npm run dev
  ```
- Client directory:
  ```
  npm run serve
  ```
